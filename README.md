### DCP Packaging Guide
To use these materials in a digital cinema environment, you must package the TIFF files into a Digital Cinema Package (DCP).

#### Tools
*   **GUI Tools:**
    *   [OpenDCP](https://opendcp.net/) (Free, Open Source)
    *   [DCP-o-matic](https://dcpomatic.com/) (Free, Open Source)
*   **Command Line Tools:**
    *   [asdcplib](http://www.cinecert.com/asdcplib/) (Standard reference implementation)
    *   `opendcp_mxf`, `opendcp_xml`

#### Basic Workflow (OpenDCP Example)
1.  **Download:** Download the TIFF file from the table above.
2.  **Convert to JPEG 2000:**
    *   Open OpenDCP -> "JPEG 2000" tab.
    *   Input: Select your TIFF file.
    *   Profile: Choose "Cinema 2K" or "Cinema 4K" matching the image resolution.
    *   Frame Rate: Set to 24 (or as required).
    *   Click "Convert" to generate `.j2c` file.
3.  **Wrap to MXF:**
    *   Go to "MXF" tab.
    *   Type: "Picture".
    *   Input: Select the `.j2c` file.
    *   Output: Save as `.mxf`.
4.  **Create DCP (XML):**
    *   Go to "DCP" tab.
    *   Title: Enter a CPL title (e.g., `DCI_White_Steps_TST_F_2K_20251129_SMPTE_OV`).
    *   Annotation: Description of the package.
    *   Picture: Add the `.mxf` file.
    *   Click "Create DCP".
