# Color Accuracy Series

## 1. 来源 (Location)
7.5.12

## 2. 用途 (Purpose)
验证SDR色彩准确性。

## 3. 特性 (Characteristics)
包含红(Red-1)、绿(Green-1)、蓝(Blue-1)三个色块。目标亮度/色度参考SMPTE-431-2 Table A.1。

## 4. 验证方法 (Verification)
### DCI合规性测试
根据CTP 7.5.12要求：
1. 使用光谱仪测量色块的亮度和色度坐标(x, y)。
2. 验证色度坐标与目标值的误差在Review Room Tolerances内。
3. 验证亮度误差在±3%以内。

### 素材文件验证
验证TIFF文件RGB/XYZ值符合目标色度坐标的转换结果。
